Anotações para Django

1 - Criar um ambiente virtual.
2 - fazer o pip install Django
2.1 - iniciar um projeto. Pode colocar qualquer nome:
    "django-admin startproject project ." --> o ponto no final é para o projeto ser criado na raiz da pasta
2.2 - Para executar o App basta digitar no terminar: "python manage.py runserver"
3 - Criar um App
3.1 - Para criar um App deve-se dar o comando: "python manage.py startapp home" --> Pode ser qualquer nome. No caso estou usando home
4.0 - Para aninhar as urls do App, você pode criar a pasta "urls.py", na pasta do App.
5.0 - Para criar uma url deve-se criar a function em 'views.home', depois parametrizar em 'urls.py' em home, ainda. E, por último, ajustar o caminho em urls.py em project.
6.0 - Em seguida, deve-se criar a pasta chamada template dentro do seu App e nessa pasta criar o arquivo ".html". No caso foi criado arquivo com o nome home.html
7.0 - Após, ir em project/settings e installed app e adicionar o nome do html.
